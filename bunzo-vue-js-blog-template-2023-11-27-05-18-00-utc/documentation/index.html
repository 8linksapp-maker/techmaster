Getting Started

Installation
  • Node.js and NPM : You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.
WordPress Installation

To get WP GraphQL API, You must have a working version of Wordpress already installed

Recommended Plugins for blog environment setup
  • WPGraphQL ( https://wordpress.org/plugins/wp-graphql/ )
  • htbunzo plugins (You will get the plugins to the downloaded folder)
Include GraphQL API
  • You must have to include WP GraphQL API.

Open your nuxt.config.js file and include your GraphQL API in apollo httpEndpoint

Install NPM Packages
  • You must have to yarn install or npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
Development server
  • Run yarn dev or npm run dev for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.
Build

With static site generation you can render your application during the build phase and deploy it to any static hosting services such as Netlify, GitHub pages, Vercel etc.
This means that no server is needed in order to deploy you application.

  • Before you build your project, please see nuxt.config.js carefully and customize in the way you want.
    Run npm run generate command to build the project. The build artifacts will be stored inside the dist directory.

Customize Component

Customize The Other Components

Customizing all the other components are so easy to use once you mastered above components. It is the same case as customizing post style.
Therefore you can just insert any props that helps you change your component.

Here is an example:

<PostStyleTwo :blog="blog" />

components: { PostStyleTwo: () => import('@/components/posts/PostStyleTwo') }
